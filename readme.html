Open Video Tracker – User Guide

Introduction

Open Video Tracker integrates FFmpeg, COLMAP, and GLOMAP to automate 3D camera tracking and point-cloud reconstruction from videos directly inside Blender.

Installation

Step 1: Download
  • Clone or download the repository
  • Locate the open_video_tracker folder
Step 2: Install in Blender
  1. Go to Edit → Preferences → Add-ons
  2. Click Install... and select the folder
Step 3: Enable the Addon

Enable “Open Video Tracker” in the Add-ons list. It will appear in the 3D Viewport sidebar.

User Interface

  • Video Path: choose your input video
  • Feature Extraction: adjust resolution, GPU, and model
  • Execution: click Track Video to run the pipeline

Workflow

  1. Prepare your footage (steady and well-lit)
  2. Configure quality and GPU options
  3. Click Track Video
  4. Import the reconstruction into Blender

Importing & Post-Processing

  • Use File → Import → COLMAP Model for manual import
  • Refine the camera animation
  • Mesh and clean up point clouds
  • Integrate into your Blender scene

Best Practices

  • Use short clips for testing
  • Enable GPU acceleration
  • Ensure good lighting
  • Keep video stable for best reconstruction
